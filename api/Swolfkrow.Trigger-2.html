Namespace
Swolfkrow

The prelude to a two-step conditional composition triggered by events of type TTriggerEvent yielded by an asynchronous workflow.

public class Trigger<TEvent, TTriggerEvent> where TTriggerEvent : TEvent

Type Parameters

TEvent

The type of the events yielded by the asynchronous workflow.

TTriggerEvent

The type of the events that trigger the composition.

Remarks

The Trigger<TEvent, TTriggerEvent> class enables the declaration of two-step conditional compositions on asynchronous workflows, by storing the information relative to the triggering condition as specified in the initial call to any of the 'When' method overloads, and exposing in turn methods that allow defining the triggered action.
