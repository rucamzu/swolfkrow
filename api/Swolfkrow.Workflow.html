Start<TEvent>(IAsyncEnumerable<TEvent>)

Starts an asynchronous Workflow<TEvent> from an asynchronous workflow.

public static Workflow<TEvent> Start<TEvent>(IAsyncEnumerable<TEvent> workflow)

Parameters

workflow IAsyncEnumerable<TEvent>

An existing asynchronous workflow.

Returns

Workflow<TEvent>

An asynchronous Workflow<TEvent> that yields the events of the given workflow.

Type Parameters

TEvent

The (base) type of the events yielded by the asynchronous workflow.

Remarks

This function serves as entry point to the DSL. It is an identity function, functionally equivalent to directly accessing the given workflow.
